# Домашнее задание к занятию «12. WebWorkers, ServiceWorkers»
[![Netology AHJ](https://github.com/Milka79rus/ahj-workers-client/actions/workflows/web.yml/badge.svg)](https://github.com/Milka79rus/ahj-workers-client/actions/workflows/web.yml)


В рамках данного проекта реализовано веб-приложение, демонстрирующее работу с фоновыми потоками в браузере (**Web Workers**) и механизмами кэширования для обеспечения оффлайн-режима (**Service Workers**).

## Что было реализовано:

### 1. Интерфейс загрузки новостей (Loading Styling)
* Реализована стильная загрузка контента с использованием мерцающих заглушек (**скелетонов**).
* Настроена интеграция с сервером на Koa.js, развернутым на платформе Amvera.
* Интерфейс корректно обрабатывает отсутствие сети и падение сервера: в случае ошибки загрузки пользователю показывается информативный баннер, а бесконечная анимация загрузки отключается.

### 2. Хешер файлов (Hasher* — Задача со звёздочкой)
* Создан виджет для вычисления контрольных сумм файлов на лету без отправки данных на сервер.
* Поддерживается загрузка файлов как через стандартное диалоговое окно, так и с помощью механизма **Drag & Drop**.
* Все ресурсоемкие вычисления (чтение файлов через `ArrayBuffer` и расчет хэшей библиотекой `crypto-js`) полностью вынесены в **Web Worker** (`hash.worker.js`). Это гарантирует, что интерфейс страницы остается отзывчивым и не зависает даже при обработке больших файлов.
* Реализована поддержка алгоритмов: **MD5**, **SHA1**, **SHA256**, **SHA512**.

### 3. Интеграция с нестабильным сервисом (Buggy Service* — Задача со звёздочкой)
* С помощью плагина `Workbox` (`workbox-webpack-plugin`) настроено кэширование статических ресурсов приложения.
* Настроена стратегия `NetworkFirst` для динамических запросов к API новостей. Если сервер недоступен (ошибка 500) или у пользователя пропал интернет, **Service Worker** автоматически перехватывает запрос и отдает последние успешно сохраненные новости из кэша, сохраняя работоспособность интерфейса.

* **Сборка:** Webpack 5 
* **Фронтенд:** Vanilla JS, HTML5 (Drag & Drop API), CSS3 (Анимации скелетонов)
* **Воркеры:** Web Workers API, Service Workers (Workbox)
* **Криптография:** `crypto-js`
* **Деплой:** GitHub Actions (автоматический билд и публикация на GitHub Pages)
